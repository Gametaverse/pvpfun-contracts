# PVPFUN

A intelligent infrastructure for Programmable Viable Products, empowering Web3 creation with seamless tools to build, deploy, and assetize PvPs in one open ecosystem. built on **BNB Smart Chain (BSC)** and compatible with other EVM networks.

## Technology Stack

- **Blockchain**: BNB Smart Chain + EVM-compatible chains  
- **Smart Contracts**: Solidity ^0.8.28  
- **Frontend**: Next.js + ethers.js
- **Development**: Hardhat, OpenZeppelin libraries
